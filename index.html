DSI Classification Methods ◆

Standard DSI

True Dynamic Strength Index using isometric squat hold as the strength baseline. Most accurate classification. Collected for 8 athletes currently.

DSI = CMJ Concentric Impulse [N·s] ÷ ISOSQT Peak Force [N]
Foundation <0.07 · Diesel 0.07–0.10 · Rabbit 0.10–0.14 · Elite ≥0.14

Modified DSI — directional only

Proxy classification using CMJ RSI-modified when no isometric baseline exists. Directional guidance until ISOSQT data is collected. Treat as approximate.

Proxy = CMJ RSI-modified [m/s]
Foundation <0.35 · Diesel 0.35–0.60 · Rabbit 0.60–0.85 · Elite ≥0.85
